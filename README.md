# Smithsonian Himalayan Bird Analysis

Ecological data analysis of Himalayan bird observation data from the UCSB-Smithsonian Scholars Himalayan Project.

## Overview
This project analyzes bird population and species richness patterns across Eastern and Western Himalayan regions. The analysis compares bird counts, species diversity, elevation, annual temperature range, and precipitation to better understand regional biodiversity trends.

## My Contributions
- Merged bird observation data with forest metadata using shared grid identifiers
- Cleaned and prepared ecological datasets for analysis
- Compared bird counts and species richness across Eastern and Western regions
- Created visualizations to communicate patterns in elevation, temperature range, and precipitation
- Presented findings as part of the UCSB-Smithsonian Scholars project

## Key Findings
- More individual birds were observed in the East than the West
- The East showed higher species richness than the West
- Elevation, annual temperature range, and precipitation were useful variables for interpreting bird distribution patterns
